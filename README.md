# Embeddable Widget with React

## Getting starterd

```shell
git clone https://github.com/fuaditrockz/embeddable-widget-example.git
```

```shell
> cd embeddable-widget-example
> npm install
```

Running for build
```shell
> npm run build
```

Running for development
```shell
> npm start
```

Testing
```shell
> npm run test
```
